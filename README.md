Initial Commit
搭建框架：

1、采用ABP框架，MUtilweb/AngularJS/WPF UI

2、采用EF模式

3、没有集成ZERO权限模块

4、源码说明：

	a) AIMS.UI---前端UI层，里面包含三个工程：
		Mutil.UI:jquery+asp.net MVC5
		AngularJS.UI:angularjs+asp.net MVC5
		WPF.UI：MVVMLight+WPF
	b）AIMS.Api---API层
	c) AIMS.Application---应用领域层
	d) AIMS.Core---核心业务层
	e) AIMS.Entity---实体层
	f) AIMS.Common---公共库管理层
	g) AIMS.Docs---文档管理层
